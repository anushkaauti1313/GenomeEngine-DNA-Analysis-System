# GenomeEngine – DNA Analysis and Mutation Detection System

## Overview

GenomeEngine is a bioinformatics-based DNA sequence analysis platform developed using C++, Python, and FastAPI. The system analyzes genomic sequences, detects mutations, performs pattern matching, and generates meaningful insights through an interactive dashboard.

The project combines advanced data structures and algorithms with visualization techniques to provide efficient genome analysis and mutation detection.

## Algorithms Implemented

### Rabin-Karp Algorithm
Efficient pattern matching using hashing techniques.

### Knuth-Morris-Pratt (KMP)
Optimized string searching algorithm for genomic pattern analysis.

### Trie Data Structure
Fast prefix-based searching for DNA sequence patterns.

### Suffix Array
Efficient substring search and sequence matching.

---

## System Workflow

1. User provides DNA sequence data.
2. GenomeEngine processes the sequence.
3. Pattern matching algorithms analyze genomic patterns.
4. Mutations are identified and classified.
5. GC content and nucleotide distribution are calculated.
6. Results are stored in JSON format.
7. Dashboard visualizes analysis results and insights.
